# TabMover

A Chrome extension to enable moving active tabs left or right using customizable keyboard shortcuts, simulating dragging and repositioning tabs by keyboard alone on Mac.

## Features

*   Move the current tab one position to the left via keyboard shortcut.
*   Move the current tab one position to the right via keyboard shortcut.
*   Customizable shortcuts from the extension options.
*   Supports macOS Chrome environment fully.

## Installation

1.  Clone this repository or download the source code.
    ```bash
    git clone https://github.com/DebdootManna/TabMover.git
    ```
2.  Open Chrome and navigate to `chrome://extensions`.
3.  Enable "Developer mode".
4.  Click on "Load unpacked" and select the `TabMover` directory.

## Usage

The default shortcuts are:
*   **Move tab left:** `Command+Shift+Left` on Mac, `Ctrl+Shift+Left` on other platforms.
*   **Move tab right:** `Command+Shift+Right` on Mac, `Ctrl+Shift+Right` on other platforms.

You can customize these shortcuts in `chrome://extensions/shortcuts`.
